Lego Pixel Art Generator

This tool is built specifically around Lego set 21226. The color palette reflects exactly what's included in that kit: a fixed set of 16 colors, with each color's quantity limited to the exact number of pieces the kit provides.

The tool will never assign more pips of a color than are actually available in the kit, even if your uploaded image would otherwise call for more of that color. Because of this, the final result is a best-fit approximation of your image within those constraints — not a pixel-perfect color match.
